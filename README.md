## 简介
这是一个用于将在线长篇文档导出为本地格式的 Chrome/Edge 浏览器扩展。可以在网页加载后，提取内容并在本地组装生成文件。

## 特性支持
- **Markdown**
- **HTML** (支持图片本地化)
- **PDF** (调用浏览器原生打印引擎)
- **DOCX** (完美保留排版和图片)

## 安装使用
1. 下载/克隆本仓库到本地
2. 打开 Chrome / Edge 浏览器的扩展程序页面 (`chrome://extensions/` 或 `edge://extensions/`)
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹即可
5. 在文档页面刷新，点击扩展栏的图标即可打开侧边栏面板
